# GlicoMais

Aplicativo desenvolvido para auxiliar no controle e acompanhamento da glicemia de forma simples e prática.

## Funcionalidades

- Registro de medições de glicose.
- Histórico completo das medições.
- Visualização dos resultados de forma organizada.
- Interface simples e intuitiva.
- Armazenamento local dos dados.

## Tecnologias Utilizadas

- React Native
- Expo
- JavaScript

## Instalação

Clone o repositório:

```bash
git clone https://github.com/lucianomacedo-info/Glicomais.git
```

Acesse a pasta do projeto:

```bash
cd Glicomais
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npx expo start
```

## Objetivo

O GlicoMais foi desenvolvido para facilitar o acompanhamento diário da glicemia, permitindo que usuários registrem e consultem suas medições de maneira rápida e organizada.
